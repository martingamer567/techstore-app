# TechStore - E-commerce en React

## Descripción del proyecto

TechStore es una Single Page Application (SPA) de e-commerce de tecnología (heladeras, televisores, audio y computación) desarrollada con React. El proyecto migra una versión previa en JavaScript vanilla hacia una arquitectura moderna basada en componentes, con manejo de estado global mediante Context API, navegación con React Router, y persistencia de datos en Cloud Firestore (Firebase).

Los usuarios pueden navegar el catálogo de productos, filtrarlos por categoría, ver el detalle de cada uno, agregarlos al carrito de compras respetando el stock disponible, y finalizar la compra generando una orden real en la base de datos.

## Herramientas utilizadas

- **React** – librería principal para la construcción de la interfaz
- **Vite** – entorno de desarrollo
- **React Router DOM** – navegación entre vistas sin recarga de página (SPA)
- **Context API** – manejo del estado global del carrito de compras
- **Firebase / Cloud Firestore** – base de datos en la nube para productos y órdenes de compra
- **React Bootstrap** – componentes de UI y estilado
- **Bootstrap** – estilos base
- **SweetAlert2** – notificaciones y alertas de usuario
